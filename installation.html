Installation

Getting started with CoreRec is simple. You can install the package directly using pip.

Install with Pip

Run the following command in your terminal to install the latest version of CoreRec from PyPI.

pip install corerec

What's Next?

Once installed, head over to our Demo section to see live examples or dive into the source code on GitHub.

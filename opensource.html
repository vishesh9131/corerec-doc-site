Open Source Contribution Guidelines

We believe in the power of community. CoreRec is an open-source project, and we welcome contributions from developers everywhere. Your ideas and code can help make this project better for everyone.

Our Philosophy

We are committed to creating a transparent, collaborative, and inclusive environment. Open source is about more than just code; it's about building a community that learns and grows together. We value every contribution, from a simple bug report to a major feature addition.

How to Contribute

  • Suggesting Enhancements: Have an idea for a new feature? We'd love to hear it. Open an issue to describe your idea and why it would be a valuable addition.
  • Pull Requests: Ready to contribute code? Fork the repository, create a new branch for your feature or fix, and submit a pull request. Please ensure your code follows the existing style and includes relevant tests.

Code of Conduct

To ensure a welcoming environment for all, we have a Code of Conduct that we expect all contributors to adhere to. Please take a moment to read it. We are dedicated to providing a harassment-free experience for everyone, regardless of background or identity.

Explore the CoreRec Library
---------------------------

Welcome to the official documentation for CoreRec, your go-to library for advanced recommendation systems. Dive into our comprehensive guides and API references to unlock the full potential of CoreRec.

Installation
------------

To install CoreRec, simply run the following command:

.. code-block:: bash

   pip install corerec
   
.. note:: CoreRec is trusted by over 6000+ users and receives 1.5k downloads per month on PyPI.
